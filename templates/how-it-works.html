Where Your Funds Are Stored

🏦 Custodial Exchange Storage

All users' funds are stored on custodial exchanges (Binance and Bybit) where actual trading activities take place.

📡 API Integration

NodeVT simply transmits these activities via API integration that accesses all account information and trading activities, displaying them for users on nodevt.com

🔗 What This Enables

This makes it possible to manage thousands of accounts uniquely with a single Binance account trading bot.

🔍 Transparency: Pool funds (AUM) on the central wallet can be viewed by all users on BscScan.com.
