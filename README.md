# NODE Volatility - Crypto Trading Platform

A professional cryptocurrency trading platform with automated yield generation and referral system.

## Features

### 🔐 Authentication
- JWT-based authentication
- 5-digit referral code system
- Secure wallet generation with encryption

### 📈 Trading
- Real-time prices from CoinGecko
- Buy/Sell with 10% node fee
- Professional trading charts
- Order book simulation

### 💰 Yield System
- 20% monthly yield on token holdings
- 720 hourly distributions per month
- Automatic yield withdrawal

### 🎯 Auto Take Profit
- Automatic sell at 20% gain
- Track purchase price
- Lock in profits automatically

### 🤝 Referral Program
- 7-generation referral tree
- 50% distribution to each level
- Commission tracking

### 💳 Wallet
- USDC deposits on BEP20 (BSC)
- Encrypted private keys
- Grand balance and yield wallet

### 🎨 Frontend
- Professional dark theme
- Responsive design
- Real-time dashboard
- Portfolio tracking

## Tech Stack

- **Backend**: Django, Django REST Framework
- **Database**: SQLite (dev) / PostgreSQL (prod)
- **Blockchain**: Web3.py, BSC (BEP20)
- **Real-time**: WebSockets (Channels)
- **Frontend**: Bootstrap, Chart.js
- **Authentication**: JWT (SimpleJWT)
